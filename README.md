## What

Outputs SQM files from valid JSON, based on output from [arma-class-parser](https://github.com/Fusselwurm/arma-class-parser)

## Installation

`npm install json-to-sqm`

## Usage

```
var JsonToSqm = require('json-to-sqm')
var res = JsonToSqm({"Mission": {}})
```

## Tests
